# 🏋️ Hybrid Running & Strength App

A lightweight, mobile-friendly hybrid training program built with pure HTML and CSS.

This project combines a 3-day strength and hypertrophy lifting program with a structured beginner-to-10K running progression. It's designed for runners who want to build endurance while continuing to get stronger in the gym without sacrificing recovery.

**Live Demo:**
https://ethanw17.github.io/hybrid-running-lifting-app/

---

## Features

* 📱 Mobile-first responsive design
* 🏋️ 3-day strength & hypertrophy program
* 🏃 10K-focused running progression
* 🔥 Dynamic warm-up and recovery routines
* 🎨 Clean minimalist UI
* 📲 Installable as a Progressive Web App (PWA)
* ⚡ Built with only HTML and CSS

---

## Weekly Structure

### Strength Training

* **Monday:** Upper Strength
* **Wednesday:** Lower Strength
* **Friday:** Hybrid Hypertrophy

### Running

* **Tuesday:** Interval Training
* **Saturday:** Long Easy Run

The running progression focuses on gradually increasing running volume while minimizing injury risk through conversational-paced efforts and structured intervals.

---

## 5-Week Running Progression

| Week  | Tuesday                                                | Saturday    |
| ----- | ------------------------------------------------------ | ----------- |
| **1** | 6 × (2 min run / 1 min walk)                           | 25 min easy |
| **2** | 5 × (3 min run / 1 min walk)                           | 30 min easy |
| **3** | 4 × (5 min run / 1 min walk)                           | 35 min easy |
| **4** | 3 × (8 min run / 1 min walk)                           | 40 min easy |
| **5** | 2 × (12 min run / 1 min walk) + easy running to 35 min | 45 min easy |

## Running Philosophy

This program emphasizes:

* 🏃 Running by effort rather than pace
* ❤️ Conversational Zone 2 aerobic training
* 🦵 Gradual musculoskeletal adaptation
* 🔥 Consistency over intensity
* 💪 Balancing lifting performance with endurance training
